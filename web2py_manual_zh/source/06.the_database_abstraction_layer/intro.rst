本章导读
===============

任何一个 web框架,如果没有 ORM 层,将无法给开发人员带来效率;
web2py 当然不会忽视这一领域的需求 ~ **DAL** 就是 web2py 中自创的又一个"轮子";
因涵盖了一般ORM 几乎所有功能，所以，本章很重，主要包含:
 
    - 声明
    - 反射
    - 查询
    - SQL
    - 一对多
    - 多对多
    - 缓存
   
想开发有DB后端的复杂应用，本章无法忽略！
